M2PP: A novel computational model for predicting drug-targeted pathogenic proteins

Developer: Shiming Wang, Jie Li and Yadong Wang
School of Computer Science and Technology, Harbin Institute of Technology, No. 92 Xidazhi Street, Nangang District, Harbin City, Heilongjiang Province, China, 150001.

The method is used to predict drug-targeted pathogenic proteins.

This code is written by R and can be run on R 3.6.2 or later.

Input data:
1)disease DO similarity.txt
2)disease MESH similarity.txt
3)target functional similarity.txt
4)target secquence similarity.txt
5)target-disease-association matrix.txt
6)target-drug-association matrix.txt
7)disease-drug-association matrix.txt

Output data:
AUROC and AUPR values of 5-fold cross-validation for five times.
